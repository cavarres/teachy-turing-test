# ENEM Turing Test

Can you tell AI-generated ENEM questions from real ones?

This is an interactive test where you're shown exam questions — some from actual ENEM exams, some generated by AI — and you have to guess which is which.

**[Take the test](https://cavarres.github.io/teachy-turing-test/)**

## About

This Turing Test is part of a proof of concept for [Teachy](https://www.teachy.com.br/) exploring AI-generated exam questions that are structurally authentic to Brazil's ENEM (Exame Nacional do Ensino Médio).

Each AI-generated question goes through a multi-agent verification pipeline: two independent AI reviewers solve the problem without seeing the answer options, and only questions where both reviewers agree on the correct answer are included.

## How it works

1. You're presented with a series of questions (real and AI-generated, shuffled)
2. For each one, you guess: **Real** or **AI-generated**?
3. At the end, you see your score and which ones fooled you
